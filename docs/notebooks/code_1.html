What is Python?

A general-purpose programming language by Guido van Rossum
Intuitive and minimal coding
Interpreted not compiled
Dynamically typed, no type declarations, data type is tracked at runtime
Automatic memory management
Spacing defines blocks of code such as control structures and blocks

Why Python?

  • Advantages
  • Ease of programming
  • Minimal time to develop and maintain codes
  • Modular and object-oriented
  • Large standard and user-contributed libraries
  • Large user community

Running Code in the Interactive Shell or Line

Python is an interpreted language and you can run simple Python expressions and statements in the interactive programming environment called the shell. In Jupyter Notebook, we will run these in a ‘cell’.

Print Function

We can also force the output of a value by using the print function. The simples for would look like this.

print(<expression>)

You can also write a print function that includes two or more expressions separated by commas.

print(<expression>, ..., <expression>)
#statements are interpreted one by one
print ("Hello World")
print ("something else")
print ("2+2")

Commenting in Python

You can add comment to your python scripts with the # symbol. These comments are not run as Python code, so they will not influence your result.

Basic mathematical operations

Python supports all of the mathematical operations you would expect. For additional resources, please see https://en.wikibooks.org/wiki/Python_Programming/Basic_Math

# Examples of Arithmetic Operator

b = 4
  
# Addition of numbers
add = a + b
  
# Subtraction of numbers
sub = a - b
  
# Multiplication of number
mul = a * b
  
# Division(float) of number
div1 = a / b

div2 = a // b

mod = a % b

print('a + b = ', add)

print('a * b = ', mul)
print('a / b = ',div1)
print('a // b = ', div2)
print('a % b = ', mod)
print('a ** b = ', p)

Input Function

The input function displays a prompt for the input. For example, the prompt can be

name = input("Enter your name.  :")

The text in parentheses (“Enter your name. :”) is an argument for the input function that tells it what to do for the promt. The string returned by the function in our example is saved by assigning it to the variable name. A variable is just a name for a value and the value can be called up through the variable name. To create a variable, use =. In Python, = means assignment, not equality. If the user enters the name “Prajjwal Panday”, the value of the variable name can be viewed as follows:

print(name)
name = input("Enter your name.  :")
print(name)
